Privacy Policy

Last updated: May 19, 2025

Information We Collect

  • Article URLs: When you hover over a link, the extension fetches the URL to retrieve its content for summarization. This URL is sent to OpenAI’s API and is not stored permanently beyond an in-memory cache during your browsing session.
  • Article Text: The extension extracts the article’s text to generate a summary. Extracted text is sent to OpenAI’s servers and cached in-memory (in your browser) for performance, but not saved to disk or sent elsewhere.
  • Allowed Sites List: You can configure which domains Smart Hover Reader runs on. This list of hostnames is stored in chrome.storage.sync so it persists across your devices but is never shared with external services.

How We Use Your Information

We use the URLs and article text solely to generate AI-powered summaries via OpenAI’s API. We do not use this information for any other purpose, nor do we sell or share it with third parties.

Browser Permissions & Scripting

Smart Hover Reader uses the following browser APIs to function securely and transparently:

  • activeTab Permission: Allows the extension to access the URL of the active tab in response to a user gesture (clicking the toolbar icon).
  • scripting API: Used to inject the content script (content.js) into the active tab only after the user clicks the icon and grants permission. This approach avoids broad host permissions up front.
  • Optional Host Permissions: On first use of a new domain, the extension will request access to that specific origin (e.g., https://example.com/*) so it can fetch and summarize the article. You may grant or deny each request.

Third-Party Services

Smart Hover Reader uses OpenAI’s API to produce summaries. For details on how OpenAI handles data, please refer to their privacy policy: https://openai.com/privacy.

Data Retention

Article summaries are cached in-memory for up to one hour to improve performance. No data is written to disk or sent to any server other than OpenAI’s API.

Your Choices

  • You can clear your browser’s cache or disable the extension at any time to remove any in-memory data.
  • You can edit or remove the list of allowed sites in the extension settings popup.
  • You can revoke any granted host permissions via your browser’s Extensions page (chrome://extensions → Details → Site access).
